# Privacy Policy - Aparajita Safety Application

**Effective Date:** February 11, 2026  
**Last Updated:** February 11, 2026

## 1. Introduction
Aparajita ("we", "us", "our", or "Application") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, process, and safeguard your personal data in compliance with the Digital Personal Data Protection (DPDP) Act, 2023.

## 2. Data Controller & Processor
- **Data Controller:** Aparajita Development Team
- **Contact:** aparajita.support@example.com
- **Jurisdiction:** India (DPDP Act, 2023 compliant)

## 3. Personal Data We Collect

### 3.1 Data Provided by You:
- Username
- Age
- Email address (optional, for Google Sign-in)
- Emergency contact details (name, phone number, relationship)
- Location data (latitude, longitude, accuracy)

### 3.2 Data Generated During Use:
- Community feed posts and replies
- Safe exit schedules
- SOS alert history
- Activity logs

### 3.3 Technical Data:
- Device information
- Browser type
- Timestamps of access

## 4. Purpose of Data Collection
Data is collected and processed for the following lawful purposes:

1. **Safety & Emergency Services**
   - Location tracking for SOS alerts
   - Emergency contact notification
   - Safe exit scheduling
   - Community safety information

2. **User Authentication & Account Management**
   - Creating and managing user accounts
   - Verifying user identity
   - Personalizing user experience

3. **Service Improvement**
   - Analyzing user behavior patterns
   - Improving application features
   - Maintaining application security

4. **Legal Compliance**
   - Complying with law enforcement requests
   - Protecting user rights and safety

## 5. Data Storage & Security

### 5.1 Storage Location:
- All personal data is stored locally on your device
- Data is encrypted using AES-256 encryption
- No personal data is transmitted to external servers (except Gemini API for AI features)

### 5.2 Data Retention:
- **Active Accounts:** Data retained while account is active
- **Inactive Accounts:** Data retained for 90 days after last login
- **SOS Records:** Retained for 1 year for safety audit purposes
- **Community Posts:** Retained for duration of account + 30 days

### 5.3 Security Measures:
- Encrypted local storage
- No plaintext password storage
- Session-based token validation
- Regular security audits
- HTTPS for any external API calls

## 6. User Rights Under DPDP Act, 2023

### 6.1 Right to Access
You have the right to:
- Request and download all your personal data
- Receive data in a machine-readable format
- **How to Exercise:** Use the "Download My Data" option in Settings

### 6.2 Right to Correction
You have the right to:
- Correct inaccurate personal data
- Update incomplete information
- **How to Exercise:** Edit your profile information in the application

### 6.3 Right to Erasure (Right to be Forgotten)
You have the right to:
- Request deletion of your personal data
- Remove your account permanently
- Data will be deleted within 30 days of request
- **How to Exercise:** Use "Delete Account" in Settings (irreversible)

### 6.4 Right to Data Portability
You have the right to:
- Request your data in a structured format
- Transfer data to another service
- **How to Exercise:** Use "Export Data" in Settings

### 6.5 Right to Withdraw Consent
You can withdraw consent by:
- Disabling specific data collection features
- Deleting your account entirely

## 7. Consent & Opt-Out

### 7.1 Explicit Consent:
By using Aparajita, you explicitly consent to:
- Collection and processing of your personal data
- Storage of data as described in this policy
- Use of data for purposes outlined above

### 7.2 Withdrawal:
- You can withdraw consent at any time via Settings
- Withdrawal does not affect past data processing

## 9. Data Breach Notification
In case of any data breach:
- Users will be notified within 72 hours
- Details of breach will be disclosed
- Remedial measures will be communicated
- Report to relevant authorities (if required by law)

## 10. Children's Safety
Aparajita is designed for users aged 13 and above. If you are under 13:
- Do not use this application
- Parents/guardians can request data deletion

## 11. Grievance Redressal Mechanism

### 11.1 Filing a Grievance:
You can file a grievance regarding:
- Data processing concerns
- Privacy violations
- Consent withdrawal issues
- Data access/deletion requests

**How to File:**
- **In-App:** Settings → Privacy & Safety → File Grievance

- **Response Time:** Within 45 days

### 11.2 Escalation:
If not satisfied with response:
- Escalate to Data Protection Officer
- Further escalation to MEITY (Ministry of Electronics & Information Technology)
